# Apps Directory

This directory contains all applications developed for Apptopia issues.

## Structure

Each application is organized as follows:

```
apps/
└── issue-[id]/
    ├── README.md
    ├── src/
    ├── tests/
    └── ...
```

Please refer to [AGENTS.md](../AGENTS.md) for complete development guidelines.
